Work Projects

Data Automation ETL

I frequently work on automating data flows into our PostGIS-enabled PostgreSQL database by building and maintaining ETL pipelines, using Python to collect and process data before loading it for geospatial analysis in QGIS or ArcGIS Pro. Automation is achieved through Cron Jobs on Linux servers, handling data from sources like NASA, NOAA, and CalFire.

Metrics Pipeline and Reporting

I was tasked with establishing a comprehensive Metrics Pipeline and Reporting system to gain insights into user interactions with our web maps. To achieve this, I designed and implemented a user logging schema within our PostgreSQL database. The project seamlessly integrated SQL, Python, Pandas, NumPy, and Matplotlib to analyze data, generate meaningful metrics, and present them through visually informative graphs.

Email Marketing Campaigns

Over six months, I orchestrated and optimized email marketing campaigns for 250,000 contacts, using HTML, CSS, Photoshop, Python, Pandas, and Excel for personalized content, audience segmentation, A/B testing, and KPI tracking. This resulted in 20 campaigns with an average open rate of 25.85% and a click rate of 4.49%.

Technical Documentation

I led the overhaul of our user activity logging database schema, reducing 22,000 unique event names to 23, and added JSONB for detailed logging. Additionally, I created various technical documents for training, volunteer, and new user reference.

Update and Maintain Company Websites

I update and maintain the company's WordPress websites, using HTML, CSS, and basic JavaScript within strict guidelines to add and modify content while adhering to SEO best practices. Additionally, I implemented Google Analytics to monitor website traffic.
